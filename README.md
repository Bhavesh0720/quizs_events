## Setup Instructions (Windows)

1. Download and extract the project ZIP file from the GitHub repository.

2. Open the extracted folder in Visual Studio Code.

3. Open the integrated terminal in VS Code  
   (Terminal â†’ New Terminal).

4. change directory:<br>
   cd quizs_events-main
   
5. Create a virtual environment:  
   python -m venv myenv

6. Activate the virtual enviourment:<br>
   myenv\Scripts\activate

7. Install project dependencies:<br>
   pip install -r req.txt

8. change the directory:<br>
   cd quizs_events

9. Start the development server:<br>
   python manage.py runserver

10. Open the application in your browser:<br>
   http://127.0.0.1:8000/
